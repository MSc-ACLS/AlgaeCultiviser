# AlgaeCultiviser  

An application for visualising, analysing, and optimising microalgae cultivation.

Live version: [AlgaeCultiviser on GitHub Pages](https://msc-acls.github.io/AlgaeCultiviser/)  

---

## Usage  

### Data Requirements  

To use the tool, you need to provide your own cultivation data in **CSV format**. Two files are required:  

1. **Data file** – typically generated by the bioreactor (minute-scale measurements).  
2. **Metadata file** – typically created manually by cultivation experts (daily observations and interventions).  

**Important notes:**  

- Both files must contain a **header row (column names)** and a **unit row**.  
- The **time format** must follow:  
  ```
  dd.MM.yyyy HH:mm:ss.SSS
  ```
  Example: `01.01.2025 01:01:01.001`  
- For productivity calculations, the **metadata file must contain a column named `Trockenmasse`** (measured dry weight in g/L).  
- The metadata file must use the same name as the data file, with the suffix `_meta`.  
  - Example: `experiment25_run1.csv` → `experiment25_run1_meta.csv`
